# 🔢 RNN Handwritten Digit Recognition

A Streamlit web app that recognises handwritten digits (0–9) using a
Recurrent Neural Network (LSTM) trained on the MNIST dataset.

## How the RNN reads images

Each 28×28 image is treated as a **sequence of 28 timesteps**, where
each timestep is a row of 28 pixel values. The LSTM processes these
rows one at a time, learning the spatial pattern across rows.

## Project Structure

```
your-repo/                      ← GitHub repo root
├── .python-version             ← Pins Python 3.11 (must be at repo root)
└── rnn_mnist/
    ├── app.py                  ← Streamlit web application
    ├── train_model.py          ← LSTM training script
    ├── requirements.txt        ← Python dependencies
    └── models/
        └── rnn_model.h5        ← Trained model (generated by train_model.py)
```

## Local Setup

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Train the model (skip if rnn_model.h5 already exists)
python train_model.py

# 3. Launch the app
streamlit run app.py
```

## Streamlit Cloud Deployment

1. Push this folder to a GitHub repository
2. Place `.python-version` at the **repo root** (not inside rnn_mnist/)
3. Go to https://share.streamlit.io → New app
4. Set **Main file path** to `rnn_mnist/app.py`
5. Deploy

## Model Architecture

| Layer | Details |
|-------|---------|
| LSTM | 128 units, return_sequences=True |
| Dropout | 0.3 |
| LSTM | 64 units |
| Dropout | 0.3 |
| Dense | 64 units, ReLU |
| Dense | 10 units, Softmax |

## Notes

- Upload a **white digit on a dark background** for best results
- The app auto-inverts images that appear to be black-on-white
- Accepts PNG, JPG, JPEG
